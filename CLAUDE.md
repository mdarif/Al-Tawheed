# CLAUDE.md

The agent guide for this repo is **[AGENTS.md](AGENTS.md)** — read it first.
It is tool-agnostic and shared by every AI assistant working here.

The durable, portable memory ("second brain") lives in:
- **[docs/gotchas.md](docs/gotchas.md)** — hard-won landmines; read before non-trivial work
- **`docs/decisions/`** — ADRs (the *why* behind past choices)

When you learn something non-obvious, append it to `docs/gotchas.md` (or add an
ADR) in the same change — see the *Memory* section of AGENTS.md.
